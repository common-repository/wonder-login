== Description == 

Easy Login and Registration features with popup.

**Features:**

1. Easy to Login and Register (as modal)
2. Well customizable
3. 100% responsive
4. provides sample login and registration pages
5. Compatible with all themes
6. Tested with latest WP version
7. Possible to replace wp-login.php with a custom "Login", and "Registration" pages
8. provides after-login/registration/logout actions

**Customization options:**

1. Add plugin's classes to your buttons to attach modal.
2. Select pages for 'After-Login actions' and 'After-Logout actions'.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Setting 'Wonder Login' to configure the plugin
